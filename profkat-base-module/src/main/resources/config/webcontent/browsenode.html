Tausende von Dokumenten ... und ständig werden es mehr....

Dabei die Übersicht zu behalten fällt schwer. Eine Hilfe bieten Klassifikationssysteme. Dokumente können verschiedenen Klassifikationen zugeordnet werden.

Durch das Browsen in einer Klassifikation ist eine hierarchisch strukturierte, thematische Suche von Dokumenten nach definierten Ordnungskriterien möglich.
